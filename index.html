GPT_Cov_Let.scrapeToGPT.completionQuery(desc)¶

Takes the description and combines it with a preset query to send to openAI.

Parameters:

desc – Description from the html of descScrape is automatically put in here. You must also enter your openAI api key.

GPT_Cov_Let.scrapeToGPT.descScrape(html)¶

Webscrapes the html description of the LinkedIn url of a job posting.

Parameters:

html – The html scraped from the urlScrape function automatically goes here.

GPT_Cov_Let.scrapeToGPT.urlScrape(url)¶

Boots up a chrome browser window and logs in. With your credentials.

Parameters:

url – Your typical www.linkin.com/… url of a typical job post. It needs your linkedin username and password.
